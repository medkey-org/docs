# Installation

Hospital information system Medkey - cross-platform application. You can install it on any modern operating system, supporting PHP. We support next operating systems installation guides: Debian, Ubuntu, CentOS Ð¸ Windows. Also we support installation using Docker, and web hosting installation.

{% hint style="info" %}
If you have your own server or virtual machine \(VPS/VDS\) - we recommend to use Linux operating systems \(Debian/Ubuntu, CentOS/RedHat\). Also Docker installation may be useful for you.

If you have resources on any hosting, then use hosting installation guide.
